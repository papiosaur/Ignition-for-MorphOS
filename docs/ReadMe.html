ignition

beta preview release, june 2003
 

Notes about this release

This is a beta release of ignition for MorphOS. For now, its user interface is completely in German, but the localisation is currently underway. Sorry for any inconvenience.

Some further remarks:

  • the TurboCalc import module is not yet finished - it doesn't support multiple pages, for example
  • Excel import/export filters are not included in this release
  • Printing and saving of documents have been disabled for this release
  • The provided documentation is only basic at this point. The next release will include HTML docs in both, English and German.

Notes about the compatibility to MorphOS

ignition was specifically adapted to run better on MorphOS, but there are still some issues left. Notably:

  • There is a bug in the picture.datatype that prevents ignition to correctly show its icons - they are all rendered using wrong colors. If this won't be fixed for the final MorphOS release, this will be worked around, though.
  • Printing has not been tested
  • You may need very recent version of locale.library, iffparse.library, and ft2.library for ignition to work correctly - if they aren't publicly available yet, it's safe to use the AmigaOS counterparts (bullet.library instead of ft2.library)
  • You should have some vector fonts installed - this beta doesn't include any fonts.
  • AmigaGuide support is still missing in MorphOS, and thus, the context sensitive help won't work at all if you don't have installed the AmigaOS version of it.
  • With very large font sizes and a deep zoom level, it can happen that texts aren't visible anymore. This seems to be an internal limit of the ft2.library - the next release might impose stricter limits on the maximum font size to hide this "feature" from you.
